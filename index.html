01 / LEARNING JOURNEY

學習歷程

01 / CLI
純 Python CLI
入門 · 無需任何套件
  • 公制 / 英制雙單位支援
  • 手動輸入驗證（if/else）
  • BMI 分類與健康建議
  • AI Agent 根據 spec.txt 自動產生
  • 單一檔案，邏輯清晰易讀
02 / FASTAPI
FastAPI 後端系統
進階 · 分層架構 · REST API
  • POST /calculate REST API 端點
  • Pydantic 自動資料驗證
  • 自動偵測身高單位（cm / m）
  • api / models / services 分層設計
  • 完整單元測試（test_api / test_service）
